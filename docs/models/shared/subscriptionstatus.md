# SubscriptionStatus

Status of subscription. Possible values are: `active`, `in_trial`, `canceled`, `paused`, `unknown`


## Values

| Name       | Value      |
| ---------- | ---------- |
| `Active`   | active     |
| `InTrial`  | in_trial   |
| `Canceled` | canceled   |
| `Paused`   | paused     |
| `Unknown`  | unknown    |
